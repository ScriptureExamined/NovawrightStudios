Beginner Website Builder

A desktop application designed to help beginners create professional websites without writing code.

Beginner Website Builder combines a visual editor, live preview system, customizable layouts, and reusable website templates into an easy-to-use development environment. Users can build complete multi-page websites using sections, images, content blocks, menus, and styling controls—all through an intuitive graphical interface.

The project was created to make website creation accessible to small businesses, hobbyists, churches, clubs, and anyone who wants a professional online presence without learning HTML, CSS, or JavaScript.

Technologies: C#, Windows Forms, WebView2, HTML, CSS, JavaScript
