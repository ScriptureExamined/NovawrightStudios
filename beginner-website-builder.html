Beginner Website Builder

Beginner Website Builder is a Windows desktop application created by NovaWright Studios to help beginners create professional websites without writing code.

The app provides a visual editing experience where users can build pages with sections, images, text, buttons, galleries, content cards, menus, and styling controls. Changes can be previewed directly inside the application before the website is generated.

Project Goals

The goal of Beginner Website Builder is to make website creation easier for small businesses, churches, clubs, hobbyists, and anyone who wants a clean online presence without needing to learn HTML, CSS, or JavaScript.

Key Features

  • Visual website editing
  • Live website preview
  • Section-based page building
  • Custom colors, fonts, spacing, and layout controls
  • Reusable sample website projects
  • Static HTML and CSS website generation

Technologies Used

C#, Windows Forms, WebView2, HTML, CSS, JavaScript, and JSON.

Development Status

Beginner Website Builder is currently in active development. Recent work has focused on improving section editing, live preview behavior, image layouts, gallery controls, and sample website projects.
