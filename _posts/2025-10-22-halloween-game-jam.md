The jam ran for 10 days, but I barely had 3 days to get my submission in. You could only use freely available assets. First jam ever using Unity, so there was a bit of a learning curve. I couldn't get it the way I wanted it, so it was basically an escape room type puzzle game with only one puzzle.

No start screen, no end screen, no music. It was probably the quickest game in the jam. I did get it submitted, so I've got that going for me.I think the ideas are fantastic, and can't wait for the jam to end so I can start adding to it and polish it.

You can find my game <a href="https://stannellc.itch.io/static-souls">here</a>.
